=====================================================================
					myanimelist-custom-list-styles
=====================================================================

Sur ce projet, vous trouverrez un ensemble de style css déjà prêts,
qui sont à utilisé sur le site Myanimelist.net, site de gestion de 
librairie de manga et d'animation japonais, pour personnaliser votre
propre list.
Les fichiers présents ici ont été récpérer principalement via le forum
de MAL puis modifier pour correspondre à mes gouts, malgrès que j'ai été
tres limité par mes connaissances en css. N'hésiter pas à les utiliser,
voir les personnaliser à votre tour!
